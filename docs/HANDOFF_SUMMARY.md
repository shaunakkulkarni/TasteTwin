# TasteTwin Handoff Summary

TasteTwin is an iOS MVP for music logging and explainable recommendations.

## What the app is
A “Letterboxd for music” experience where users:
- search albums
- log ratings/reviews/tags
- build a listening history
- eventually get an AI Taste Twin profile
- receive explainable recommendations with receipts from prior logs

## Technical direction
- SwiftUI
- SwiftData
- MusicKit
- Apple Foundation Models
- layered MVVM
- repositories + services
- local-first MVP
